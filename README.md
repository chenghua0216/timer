# timer

此儲存庫包含兩個單一檔案的網頁 App：

- `index.html` — 開會計時器
- `whiteboard.html` — 無邊白板：類似 Apple「無邊記 (Freeform)」的無限畫布白板

## 無邊白板

直接用瀏覽器開啟 `whiteboard.html` 即可使用（支援手機觸控與桌面滑鼠）。

功能：

- 無限畫布：拖曳平移、雙指縮放（桌面可用 Ctrl+滾輪）
- 畫筆：多種顏色與粗細、橡皮擦
- 文字、便條紙、形狀（矩形／圓角／橢圓／三角形／星形／直線）、表格、照片
- 物件選取、拖移、縮放、換色、複製、刪除
- 還原／重做（Cmd/Ctrl+Z）
- 內容自動儲存於瀏覽器（localStorage）
- 匯出整個白板為 PNG 圖片
